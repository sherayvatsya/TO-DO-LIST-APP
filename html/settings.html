    About App

    TaskFlow is a local-first productivity dashboard with authentication, task planning, Pomodoro focus, analysis, themes, backups, and keyboard shortcuts.

    Shortcuts: Enter adds tasks, Ctrl + F searches, Delete removes the selected task, Ctrl + D toggles dark mode.
